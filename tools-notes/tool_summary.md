## Burp Suite

Used to capture HTTP traffic during exploitation.
Confirmed active communication between attacker and target systems.
Ensured evidence integrity through packet inspection and hashing.

## Draw.io
Used to visualize the exploit chain and attack path from initial access to root compromise.
